# java-microservices
Java Spring-boot Micro-services blueprint


* Spring-boot fresh build
* docker
* open API
* make file
* message broker
* layer architecture (DDD)
* CI/CD
* auth service
* email service, main-service
* kubernetes 

## Setup: 
### run:
* make build
* make run
* to reload the Service use make reload
