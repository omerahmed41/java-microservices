# java-microservices
Java Spring-boot Micro-services blueprint


* Spring-boot fresh build
* docker
* open API
* make file
* Logs slf4j
* message broker
* auth service
* email service, main-service
* Service registry
* API-gateway
* Design pattern
* layer architecture (DDD)
* CI/CD
* kubernetes



## Setup: 
### run:
* make build
* make run
* to reload the Service use make reload
